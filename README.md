## Descripción del proyecto:
La aplicación es un Gestor de Proyectos que permite:
- Agregar proyectos y el estado del proyecto.
- Eliminar proyectos registrados.
- Buscar proyectos en tiempo real.
- Visualizar los proyectos registrados en tiempo real.

Esta herramienta brinda organización y comodidad a la hora de gestionar proyectos.
